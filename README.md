# My2048

这是一个用C++写的破产版2048，也是本人独立写的第一个小项目~

## 项目背景

- 2048是一个风靡全球的益智类小游戏，通过上下左右控制来合并盘子里的数字，直到盘子里出现2048。

## 开发环境

- Windows 10 64位 ( DirectX 12 )
